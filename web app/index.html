Automatic Greenhouse

take care of your plants without effort

This is the individual project for the course of Internet of Things 2021/2022 of the Master in Computer Science, La Sapienza Università di Roma.
The idea behind this project is to create a smart greenhouse for plants that can measure the environment in which the plant is and act depending on the values it sense. All the values registered by the sensors are sent to AWS where are processed and stored. A user once on this web interface not only can access the most recent stored values but it is also notified in real time of the values that are being sensed, moreover from this web interface the user can simply interact with the actuator directly influencing their status.
More detailed information and code can be found in the followung github repository: automaticGreenhouse Repository.
